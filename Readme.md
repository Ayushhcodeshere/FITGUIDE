# 🏋️ FITGUIDE – Fitness Guide Web App

FITGUIDE is a simple fitness web application built using **Python Flask**.  
This project helps users enter basic details like **name, age, and fitness goal**, and then redirects them to a personalized dashboard page.

It is beginner-friendly and made for learning **Flask web development**.

---

## ✨ Features

✔️ Simple and clean user interface  
✔️ User enters details like name, age, and goal  
✔️ Personalized dashboard display  
✔️ Built using Flask, HTML, CSS, and JavaScript  
✔️ Beginner-friendly project

---

## 🚀 How to Run This Project

Follow these simple steps:

### Step 1: Clone the Repository

```bash
git clone <https://github.com/Ayushhcodeshere/FITGUIDE.git>
```

Example:

```bash
git clone https://github.com/Ayushhcodeshere/FITGUIDE.git
```

---

### Step 2: Open Project Folder

```bash
cd FITGUIDE
```

---

### Step 3: Install Flask

If Flask is not installed, run:

```bash
pip install flask
```

---

### Step 4: Run the Project

Run this command:

```bash
python app.py
```

---

### Step 5: Open in Browser

After running, open:

```bash
http://127.0.0.1:5000/
```

Now your project will run successfully 🎉
